로또픽 개인정보 처리방침

최종 업데이트: 2026년 5월 15일

로또픽 앱은 사용자의 개인정보를 수집·저장·판매하지 않습니다. 앱에서 사용하는 모든 데이터는 사용자의 기기에만 저장됩니다.

1. 수집하는 정보

로또픽은 다음 정보를 기기 내에서만 처리하며 외부 서버로 전송하지 않습니다.

  • 위치 정보: 주변 복권 판매점 찾기 기능 사용 시 현재 위치를 일시적으로 사용합니다. 위치 정보는 저장되지 않으며 지도 앱 실행 후 즉시 폐기됩니다.
  • 카메라: 복권 QR코드 스캔 기능에만 사용됩니다. 촬영된 이미지는 저장·전송되지 않습니다.
  • 알림: 사용자가 직접 설정한 주간 추첨 알림 일정은 기기 내에만 저장됩니다.
  • 앱 사용 데이터: 생성한 번호, 저장 번호, 필터 설정 등은 기기 내 로컬 저장소(AsyncStorage)에만 저장됩니다.
  • 로또 당첨 데이터: 동행복권(dhlottery.co.kr) 공개 API에서 당첨 번호를 조회하여 기기 내 SQLite DB에 저장합니다. 사용자 식별 정보는 포함되지 않습니다.

2. 정보 제3자 제공

로또픽은 사용자 정보를 제3자에게 제공, 공유, 판매하지 않습니다.

3. 외부 서비스 연동

  • 동행복권 API: 로또 당첨 번호 조회를 위해 동행복권 공개 API를 사용합니다. 이 과정에서 사용자 개인정보는 전송되지 않습니다.
  • Google Maps / Apple Maps: 주변 판매점 찾기 시 지도 앱으로 연결됩니다. 해당 앱의 개인정보 처리방침이 적용됩니다.

4. 권한 사용 목적

  • 카메라 권한: 복권 QR코드 스캔 당첨 확인 전용
  • 위치 권한: 주변 복권 판매점 검색 전용 (사용 후 즉시 폐기)
  • 알림 권한: 사용자 요청 시 주간 추첨 알림 발송 전용

5. 데이터 보관 및 삭제

모든 데이터는 사용자의 기기에만 저장됩니다. 앱 삭제 시 모든 데이터가 함께 삭제됩니다.

6. 아동 개인정보 보호

로또픽은 만 14세 미만 아동으로부터 개인정보를 의도적으로 수집하지 않습니다. 본 앱은 성인용 복권 정보 제공 앱입니다.

7. 면책사항

로또픽은 정보 제공 목적으로만 제공되며 당첨을 보장하지 않습니다. 로또는 사행성 게임으로 과도한 구매를 자제해 주시기 바랍니다.
