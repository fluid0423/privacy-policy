공휴일 병원/약국 앱 · 시행일: 2025년 1월 1일

1. 수집하는 개인정보 항목

본 앱은 서비스 제공을 위해 아래 정보를 수집합니다.

  • 위치정보: 주변 병원·약국 검색에만 사용됩니다.
  • 수집된 위치정보는 기기 외부로 전송되거나 저장되지 않습니다.

2. 개인정보의 이용 목적

  • 현재 위치 기반 주변 병원·약국 검색
  • 검색 결과 제공 및 거리 계산

3. 개인정보의 보유 및 이용 기간

위치정보는 검색 요청 시 일시적으로 처리되며 별도로 저장하지 않습니다. 즐겨찾기 데이터(병원 ID만)는 앱을 삭제하면 완전히 삭제됩니다.

4. 개인정보의 제3자 제공

수집된 개인정보는 제3자에게 제공하지 않습니다. 단, 검색을 위해 다음 공공 API를 활용합니다.

  • 건강보험심사평가원 병원정보서비스 (공공데이터포털)
  • 국립중앙의료원 전국 응급의료기관·약국 정보 (공공데이터포털)
  • 카카오 로컬 API (좌표 변환 목적)

5. 이용자의 권리

앱 삭제 시 기기 내 저장된 모든 데이터(즐겨찾기)가 삭제됩니다.
